modParser::collectElementTags

Collects element tags in a string and injects them into an array.

Syntax

 API Doc: http://api.modx.com/revolution/2.2/db_core_model_modx_modparser.class.html#\modParser::collectElementTags() 

integer collectElementTags (string $origContent,  &$matches, [string $prefix = '[['], [string $suffix = ']]'])

The &$matches array is multi-dimensional: each element tag collected is stored in $matches as a two-item array:

  • $match[0] - the full tag (i.e. [[!$myChunk? &arg=`foo`]])
  • $match[1] - just the inner contents of the tag, without the outer brackets (i.e. !$myChunk? &arg=`foo`)
