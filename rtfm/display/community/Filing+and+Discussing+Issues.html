Please file issues using the issue tracker at http://bugs.modx.com

Issues that the community can initiate or collaborate on include

  • bugs in MODX (Evolution and Revolution) and some Extras
  • feature requests for MODX (Evolution and Revolution) and some Extras
  • reports of bugs or problems with various MODX websites or parts thereof

These are organized into Projects in the issue tracker. See a full list of them.

Many users also discuss bugs in the forums, under an appropriate thread. If you are filing a bug report for which there is forum discussion, please include a link to the forum thread.

There is no reason, however, not to use the issue tracker as a discussion forum on the problem at hand. Dialog is often very helpful especially when the solution to a bug is not clear-cut. Ultimately, coders working on bug fixes will use the issue tracker as the first, and possibly only, source of information on the problem. If there is no link to additional information on the bug, that information may never play a role in solving the bug.
