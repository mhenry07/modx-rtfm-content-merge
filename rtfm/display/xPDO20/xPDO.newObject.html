xPDO::newObject

Creates a new instance of a specified class.
All new objects created with this method are transient until xPDOObject::save() is called the first time and is reflected by the xPDOObject::$_new property.

Syntax

API Docs: http://api.modxcms.com/xpdo/xPDO.html#newObject 

$box = $xpdo->newObject('Box');
$box->set('width', 10);
$box->save();

Mass Assignment

You can set an object's attributes at the time of instantiation (referred to as "mass assignment"). 

$box = $xpdo->newObject('Box',array(
   'width' => 10,
   'height' => 4,
));
$box->save();
