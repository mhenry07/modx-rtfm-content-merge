xPDOObject::toJSON()

Returns a JSON representation of the object.

Syntax

API Docs: http://api.modxcms.com/xpdo/om/xPDOObject.html#toJSON 

    
$object->set('name','Bob');
$object->set('email','pinkdaisies@gmail.com');
$json = $object->toJSON();
echo $json;
// prints {"name":"Bob","email":"pinkdaisies@gmail.com"}
