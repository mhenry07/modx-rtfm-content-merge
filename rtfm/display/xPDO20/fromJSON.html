xPDOObject::fromJSON()

Sets the object fields from a JSON object string.

Syntax

API Docs: http://api.modxcms.com/xpdo/om/xPDOObject.html#fromJSON 

    
$str = '{"name":"Sirius","email":"Black"}';
$object->fromJSON($str);
echo $object->get('name').' '.$object->get('email');
// prints "Sirius Black"
