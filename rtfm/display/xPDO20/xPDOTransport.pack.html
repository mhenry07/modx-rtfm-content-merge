xPDOTransport::pack

Pack the xPDOTransport instance in preparation for distribution. This packs the transport into a zip file in the target directory.

Syntax

API Docs: http://api.modxcms.com/xpdo/transport/xPDOTransport.html#pack 

boolean pack ()

Example

Package the transport into a zip file. 

$transport->pack();
