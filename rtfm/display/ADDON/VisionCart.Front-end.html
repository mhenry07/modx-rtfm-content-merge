In the true MODX spirit, the front-end is completely customizable to your liking. The VisionCart package comes with a load of snippets you can use to present your shops, products, orders and more. Below you will find an overview of them and their main use. If documentation is available for the snippet, the name will link you there.

WIP - please do add!

Snippet name Usage
vcExport  
vcFormItGetUser A hook for FormIt to get information about the currently logged in user. Will send to the unauthorized page if no user has been logged in.
vcFormItSaveUser A hook for FormIt that updates the current user and returns the reponse.
vcGetBasket Displays the shopping basket and a checkout / empty basket button.
vcGetCategories Lists the different categories in a shop.
vcGetCategory Returns a single category.
vcGetProduct Returns a single product.

vcGetProductLists  
vcGetProducts Lists the different products. Can include SKUs aswell.
vcOrderHistory  
vcOrderProcess Main snippet to use for the order process. Will load the proper processor depending on that current status.
