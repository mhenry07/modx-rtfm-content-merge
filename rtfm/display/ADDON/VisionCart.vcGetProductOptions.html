What is vcGetProductOptions?

vcGetProductOptions is a snippet that gets the options of a VisionCart product and displays them using a chunk or config-file (included in the theme)

How to add product options

  1. Create an option list in your shop
  2. Go to the product
  3. Go to the Attributes tab
  4. Add the option and value
  5. Go to the SKU's tab
  6. Create a new SKU and make sure it's activated
  7. Go back to the Attributs tab
  8. Select another value from the option list(s)
  9. Keep adding SKU's for every option/value combination you need

Display product options

  1. Go to your option list
  2. Right click and choose Update
  3. Select the correct output filter (example: vcOptionOutputCombo)
  4. Make sure the vcGetProductOptions is somewhere on the product page
