What is album.view?

This snippet displays the selected album information and the album items.

Usage

Example for album.view: 

[[!album.view]]

You can also specify the templates: 

[[!album.view? &tplAlbum=`album.view` &tplAlbumItems=`album.items.list`]]

See the snippet properties for more options.

Properties

Name Description
Default
Version
tplAlbum The container to use to hold the album information and the album items.
album.view >0.0.1

album.items.list >0.0.1
tplAltItems A list of alternate image links that are defined by the image sizes in the manager that is not the primary image.
album.items.alt >0.0.1
