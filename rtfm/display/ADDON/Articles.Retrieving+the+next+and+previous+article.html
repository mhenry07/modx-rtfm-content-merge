Retrieving the next and/or previous article

To retrieve the next or previous article, again, we can use the getResources addon. In the forums I found a link to a great article how to do that. Basically, you use the current template and the creation date as where parameters to retrieve the next- and previous articles.

http://www.cmslesstraveled.com/index/archive/2010/linking-to-the-next-or-previous-document-by-publication-date.html

Remember, you can use placeholders, so you can check if you got any results using an output modifier.
