Examples for EventsX

Display a list of upcoming events

Go to components -> EventsX and create some events

(make sure they are active)

create your chunk "upcomingEvents": 

    
        <h2>next events:</h2>
        <ol id="eventcal">
                [[!EventsX? &tpl=`upcomingEvents` &limit=`10`]]
        </ol>
