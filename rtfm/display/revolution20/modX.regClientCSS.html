modX::regClientCSS

Register CSS to be injected inside the HEAD tag of a resource.

Syntax

API Doc: http://api.modx.com/revolution/2.2/db_core_model_modx_modx.class.html#%5CmodX::regClientCSS()

Example

Register a CSS file to the HEAD tag:
