friendly_alias_translit

Name: FURL Alias Transliteration
Type: textfield
Default: none
Available In: Revolution 2.0.8+

The method of transliteration to use on an alias specified for a Resource. Empty or "none" is the default which skips transliteration. Other possible values are "iconv" (if available) or a named transliteration table provided by a custom transliteration service class, specified in friendly_alias_translit_class.
