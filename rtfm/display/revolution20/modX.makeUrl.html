modX::makeUrl

Generates an URL to the specified resource. The first parameter (resource ID) is required, the rest are optional. This method is called internally for link tags ([[~5]]).

Parameter Description Default
$id Required. The ID of a resource to create a link for.
$context Optional. Specifies a context the resource. Specifying the context is optional but can save a database lookup if the resource is in a different context. Empty string, interpreted as current context.
$args Optional. A query string or array of arguments to add to the link.
$scheme Optional. The URL scheme for the generated url, see the options below. -1
$options An array of options to override in the url generation. This can override settings like friendly_urls, site_start, site_url, base_url, use_weblink_target, request_controller, request_param_id, https_port, though there are not many use cases to override these so only use if you know what you're doing.

The scheme indicates in what format the URL is generated.

  • -1: (default value) URL is relative to the site_url. Example: services/webdesign.html
  • full: URL is absolute, prepended with site_url from config. Example: http://siteurl.com/services/webdesign.html
  • abs: URL is absolute, prepended with base_url from config. Example: /services/webdesign.html
  • http (or 0): URL is absolute, forced to http scheme. Example: http://siteurl.com/services/webdesign.html
  • https (or 1): URL is absolute, forced to https scheme. Example: https://siteurl.com.com/services/webdesign.html

Usage

API Doc: http://api.modx.com/revolution/2.2/db_core_model_modx_modx.class.html#%5CmodX::makeUrl()

Makes an URL to the resource with ID 4. 

<span  style="font-family: Menlo, Monaco, monospace, sans-serif; white-space: pre;">$url = $modx->makeUrl(4);</span>

Make a URL for the Resource with ID 12, but make sure it's in HTTPS. 

$url = $modx->makeUrl(12, '', '', 'https');

Add some URL parameters to the generated url (both examples have the same result). 

$url = $modx->makeUrl(25, '', array('hello' => 'world'));
$url = $modx->makeUrl(25, '', 'hello=world');

Note that the arguments available to this function can be passed to the [[~link]] tags, e.g. 

[[~123? &scheme=`full`]]
