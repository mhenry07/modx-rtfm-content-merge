friendly_alias_translit_class

Name: FURL Alias Transliteration Service Class
Type: textfield
Default: translit.modTransliterate
Available In: Revolution 2.0.8+

An optional service class to provide named transliteration services for FURL Alias generation/filtering. A path to it can be specified by the friendly_alias_translit_class_path setting; otherwise it defaults to the normal path.
