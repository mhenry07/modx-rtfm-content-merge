modUser::removeSessionCookie

Syntax

API Doc: http://api.modx.com/revolution/2.2/db_core_model_modx_moduser.class.html#%5CmodUser::removeSessionCookie()

Example

Remove the Session Cookie for the User in the 'sports' Context.
