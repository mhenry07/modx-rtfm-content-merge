xPDOQuery::groupby

Add a GROUP BY clause to the query.

Syntax

API Docs: http://api.modxcms.com/xpdo/om/xPDOQuery.html#groupby 

    $query = $xpdo->newQuery('Box');
$query->where(array(
   'width' => 15,
));
$query->groupby('type');
$boxes = $xpdo->getCollection('Box',$query);
