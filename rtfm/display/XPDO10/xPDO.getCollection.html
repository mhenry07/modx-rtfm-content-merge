xPDO::getCollection

Retrieves a collection of xPDOObjects by the specified xPDOCriteria.

If none are found, returns an empty array.

Syntax

API Docs: http://api.modxcms.com/xpdo/xPDO.html#getCollection 

    $boxes = $xpdo->getCollection('Box',array(
   'width' => 40,
));
