DocumentParser ($modx)

The DocumentParser class is the core of the MODx page rendering framework. The DocumentParser object can be accessed with the global $modx from Snippets, Modules, and Plugins.

To see what the $modx variable contains when your script is run, try putting this in to your script: 

<?php
die(print_r($modx,true));
?>
WARNING!
That command will output a large amount of data to your browser, it also should not be executed on live sites as doing so reveals your database password. Use at your own risk! Also keep in mind that the $modx contains your database username and password and executing the line above will expose those credentials.
